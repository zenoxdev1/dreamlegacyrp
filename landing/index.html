Access flow

How to Join

The steps below keep the community organized and make sure each player is ready before joining.

Requirements

  1. PlayStation 3 console.
  2. Custom Firmware (CFW) or HEN installed.
  3. A valid Discord account.
  4. A new PSN account for roleplay.
  5. A new GTA Online character.

Enlistment

  1. Join the Dream Legacy RP Discord server.
  2. Complete the required verification.
  3. Log in on this site with Discord.
  4. Once verified, you are ready to connect.
Must read

Server Rules

Read this before you connect. Breaking any of these is an immediate and final ban.

Server Rules

  1. You must use the GTA:ODT Services.
  2. No .sprx other than the one currently in use (chaos modz).
  3. No plugins, RCE, or RTM.

RP In-Game Rules

  1. No freezing, crashing, or kicking other players without permission.
  2. No abuse mods: god mode, vehicle upgrade, no-clip, etc.
  3. Avoid unrealistic movement: bunny hopping, running everywhere, super jumps, or glitches.
  4. No metagaming — don't use outside information (like voice recognition) inside the RP.
  5. NLR (New Life Rule): if you die, you must forget past events.
  6. No free killing — killing without a proper reason is not allowed.

Mod Menu Setup

Open the menu and enable the following:

  1. Wanted level: 0 lock, so AI police won't disturb scenes.
  2. Off the radar — like FiveM, you won't see other players on the map (avoids metagaming too).
  3. Check the Animations tab and note the most useful ones.
  4. The menu will keep getting updated to be more RP-friendly.

Bug / Lag Excuses

In-character cover stories for real-life issues:

  • Lag: "I'm tired"
  • Microphone problem: "I'm sick" / voice issue / chewing gum
  • DM Groups: "Radio"
  • Server Groups: "Near me"
  • Disconnect: "Passed out"
  • Glitch: "Black magic"
  • Other: Make up an excuse, or ignore it.

Banned Assets — Only Use What's Yours

Everything in-game must match what's actually on your account:

  1. In-game weapons: only what's in your Assets tab on the website.
  2. In-game money: only what's in your Bank tab on the website.
  3. Mod menu / invisible clothing: only clothes from stores, paid for with your money.
  4. In-game phone: only the phone you bought in the DreamOS tab.
  5. Modded vehicles: only vehicles you bought or rented.
  6. Op / Job assets: no military, police, or OP weapons.
  7. Skins / models: no weird skins or animal models unless given permission.
  8. Mod menu spawned assets: don't spawn anything using your menu.
Breaking any of these rules results in an immediate and final ban. Questions? Ask in #general-chat.
